# Orchivillando Studio Brand Guide

## Positioning

Orchivillando Studio is a premium indie game studio building polished browser games, mobile experiences, and AI-powered game development tools.

## Voice

Confident, focused, cinematic, practical, and player-first.

## Colors

- Background: `#050816`
- Primary: `#6C63FF`
- Secondary: `#00D4FF`
- Accent: `#FFB703`
- Success: `#5BE37D`
- Text: `#F7F8FF`
- Muted text: `#AEB6D9`

## Typography

Use system UI fonts with strong display weight and generous line height:

`Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

## Logo

Use the orchid-play mark for studio identity. Keep clear space equal to the icon padding. Use the dark logo on dark surfaces and the light logo on bright surfaces.
